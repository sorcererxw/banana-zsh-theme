## Usage

1. `wget -O ~/.oh-my-zsh/themes/banana.zsh-theme https://raw.githubusercontent.com/sorcererxw/banana-zsh-theme/master/banana.zsh-theme`
2. change `ZSH_THEME` in `~/.zshrc` to `"banana"`
3. `source ~/.zshrc`
